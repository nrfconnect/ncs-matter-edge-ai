|addon| for the |NCS|
#####################

The |addon| for the |NCS| provides support for developing the Matter protocol with Edge AI support.
Additionally, the |addon| includes the nRF Edge AI library (version |edge-ai_version|).
In combination with the |NCS|, the |addon| allows for development of Matter Edge AI applications.
