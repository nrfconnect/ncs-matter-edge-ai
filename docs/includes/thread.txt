If you want to commission the |matter_name| device and control it remotely through a Thread network, you need to set up the `Thread Border Router <Thread tools_#ug-thread-tools-tbr>`_ and control it with the `chip-tool`_, or use a commercial ecosystem controller.
You can also control it using a Matter controller `configured on PC or smartphone <Testing Matter in the NCS_>`_.
This requires additional hardware depending on the setup you choose.
